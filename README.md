# Ideal Robot — Bot Command Center

A cyberpunk-styled React dashboard for monitoring and controlling your Telegram bot fleet.

## Features

- **Real-time status**: Monitor all 3 bots (Alexis, Three Easy, Dr. Leah)
- **System health**: Live uptime and performance metrics
- **Personality editor**: Edit bot voices, greetings, and proactive messages
- **Dark mode UI**: Cyberpunk aesthetic with neon accents

## Tech Stack

- React 18 + TypeScript
- Vite
- Tailwind CSS
- Framer Motion
- Lucide Icons

## Setup

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

Deploy the `dist/` folder to Vercel, Netlify, or GitHub Pages.
